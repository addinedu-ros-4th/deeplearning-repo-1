# 📖 Introduction
이 프로젝트에서는 학습한 DL/ML 기술들을 활용하여 작업에 필요한 부품을 구분하고 조립공정을 진행하는 작업자의 작업 생산성을 높이고 
작업을 돕기 위한 서비스를  구현을 하였다
